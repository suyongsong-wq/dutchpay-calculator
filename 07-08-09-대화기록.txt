- 09-palette 폴더 생성, single-react-dev 에이전트로 펑크 스타일 팔레트 생성기 제작
- 저장 경로: /Users/suyong/무제 폴더 2/09-palette/index.html

▶ 구현 내용 (기능)
- HEX 텍스트 입력(#3B82F6)과 컬러 피커(<input type="color">) 양방향 동기화
  (둘 다 공유 state를 쓰므로 한쪽 변경 시 다른 쪽 자동 갱신)
- 기준 색에서 명도(Lightness)를 [-28,-14,0,+14,+28] 단계로 조절해
  밝은/어두운 변형 5단계 생성 (중앙이 기준색, 5~95% clamp)
- HEX↔RGB↔HSL 변환 함수 직접 구현
  (hexToRgb, rgbToHex, rgbToHsl, hslToRgb)
- "같은 정보, 다른 형태" 컨셉: 각 스와치를 HEX / RGB / HSL 세 형태로 동시 표시
  (배경색에 따라 글자색 자동 대비)
- 클릭 시 navigator.clipboard로 복사 + "복사됨!" 토스트 + 흔들림 애니메이션
  ("전체 복사" 버튼으로 5개 HEX 일괄 복사, textarea 폴백 포함)
- 검증: #유무, 3자리/6자리 모두 허용, 잘못된 값은 한국어 에러
- 반응형(1/2/5열), 한국어 UI

▶ 구현 내용 (펑크 디자인)
- 검은 배경(점 텍스처) + 네온 핑크/그린/옐로우/시안 UI 액센트
  (생성된 팔레트 색 자체는 사용자가 고른 실제 색 사용)
- 두꺼운 검정 테두리, 흐림 없는 하드 오프셋 섀도우, 비대칭/기울임 카드,
  버튼 눌림(press-down) 효과
- Google Fonts: Anton / Archivo Black / Bebas Neue / Space Mono
- CSS keyframes: 타이틀 이중 글리치, 마퀴 스트립, 복사 시 shake

▶ 참고
- navigator.clipboard는 보안 컨텍스트가 필요 → localhost/HTTPS에서 동작.
  file://로 직접 열면 복사가 막힐 수 있어 로컬 서버 사용 권장 (textarea 폴백 존재)
