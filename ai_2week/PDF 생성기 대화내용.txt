----------------------------------------
2) AI 작업 내용 (1차 제작)
----------------------------------------
생성 파일: [Transform] PDF 생성기.html

[기술 스택]
- React 18 + Babel Standalone (CDN)
- Tailwind CSS (CDN)
- Pretendard 한글 웹폰트
- html2canvas 1.4.1
- jsPDF 2.5.1

[한글 처리 핵심]
- jsPDF는 기본적으로 한글을 지원하지 않음
- 폰트 임베드 대신 "html2canvas + jsPDF 캡처 방식" 사용
- 화면 밖(off-screen)에 실제 A4 크기 DOM 노드를 렌더링하고,
  html2canvas로 이미지 캡처 → jsPDF에 삽입
- 즉 DOM을 그대로 래스터화하므로 한글이 깨지지 않음 (WYSIWYG)
- 캡처 전 document.fonts.ready로 웹폰트 로딩 대기

[주요 기능]
- 제목 / 본문 입력 에디터 (+ 글자수 카운터)
- 실시간 미리보기 (우측, A4 비율 축소 표시)
- 서식 옵션:
  · 글꼴 (고딕 / 명조)
  · 여백 (좁게 / 보통 / 넓게)
  · 정렬 (왼쪽 / 가운데 / 오른쪽 / 양쪽)
  · 제목 크기 슬라이더 (18~48px)
  · 본문 크기 슬라이더 (11~28px)
  · 줄 간격 슬라이더 (1.2~2.4)
  · 제목 색상 / 본문 색상
  · 오늘 날짜 표시 토글
- A4 자동 다중 페이지 분할 (내용이 길면 여러 페이지로)
- 고화질 캡처 (scale: 2)
- 안전한 파일명 자동 생성
- 모든 처리는 브라우저 안에서만 수행 (서버 전송 없음)
