주요 기능:
- 소득 입력: 종합소득금액 직접 입력 / "수입 − 필요경비" 방식 토글
- 소득공제: 인적공제(+/− 버튼, 1인당 150만원), 연금보험료·보험료·기타 공제
- 세액공제: 근로소득세액공제, 자녀세액공제, 기타
- 2024년 8단계 누진세율 적용 (6%~45%, 누진공제 방식)
- 계산 결과: 과세표준 → 산출세액 → 결정세액 → 지방소득세(10%)
  → 총 납부세액 → 실효세율 단계별 표시
- 누진세율 시각화: 과세표준이 구간별로 어떻게 과세되는지 막대그래프로 표현
- 글래스모피즘 + 토스 스타일, 반응형, 천단위 콤마 + ₩ 자동 포맷

검증 결과:
- 누진세율 계산: 과세표준 0원~15억원까지 13개 케이스 전부 일치 (OK)
- 세율표 정확도: 2024년 기준 세율·누진공제액 모두 정확
  예) 과세표준 5,000만원 → 산출세액 624만원 (5,000만×15% − 126만) 일치
- 종합 시나리오 (소득 6천만/소득공제 1천만/세액공제 50만):
    과세표준 50,000,000
    산출세액 6,240,000
    결정세액 5,740,000
    지방소득세 574,000
    총 납부세액 6,314,000
    실효세율 10.52%
- 엣지 케이스 방어 정상:
    소득 0원 → 전부 0
    소득공제 > 소득 → 과세표준 0 (음수 방지)
    세액공제 > 산출세액 → 결정세액 0 (음수 방지)

결론: 계산 로직, 세율표, 지방소득세, 실효세율, 음수 방어 모두 정상 작동.
