----------------------------------------
[1] 최초 요청 — 짤 생성기(밈 메이커) 제작
----------------------------------------
요청:
  "파일명 [Transform] 짤 생성기 (밈 메이커)
   이미지 위에 텍스트 배치 → Canvas로 다운로드 가능한 이미지 생성"

구현 결과:
  ▪ 이미지 입력 (4가지 방식)
     - 파일 업로드
     - 이미지 URL 붙여넣기 (crossOrigin="anonymous")
     - 드래그 앤 드롭
     - 클립보드 붙여넣기 (Ctrl/⌘+V)
     - 내장 샘플 배경 6종 (그라데이션 4 + 흰/검 단색)
  ▪ 텍스트 오버레이 (클래식 밈 스타일)
     - 여러 개의 독립 텍스트 레이어 (기본: 상단 + 하단)
     - 내용 편집(줄바꿈 지원), 폰트 크기(12~200px)
     - 글자색, 외곽선 색/두께
     - 폰트: Impact 대체(Anton), Noto Sans KR, Black Han Sans, Do Hyeon (한글 안전)
     - 정렬, 볼드 토글
     - 캔버스에서 직접 드래그로 위치 이동 + 위/중앙/아래 빠른 배치
  ▪ 캔버스(<canvas>)가 결과물의 기준 → 미리보기 = 다운로드 (WYSIWYG)
  ▪ 다운로드: PNG / JPG 선택 후 toDataURL()로 내보내기
  ▪ CORS 처리: tainted 캔버스 감지 시 경고 표시
  ▪ 한국어 UI, 반응형 2단 레이아웃(데스크톱) / 스택(모바일)


----------------------------------------
[2] 이모티콘(이모지) 기능 추가
----------------------------------------
요청:
  "이모티콘도 넣을 수 있도록 제작해줘"

구현 결과:
  ▪ 이모지 스티커 패널 신규 추가
     - 5개 카테고리: 표정 / 손동작 / 하트·심볼 / 동물 / 사물 (총 100여 개)
     - 클릭하면 캔버스 중앙에 스티커로 추가
  ▪ 기존 텍스트 레이어 시스템에 통합 (type: "emoji")
     - 캔버스에 컬러 이모지로 렌더링 (Apple Color Emoji / Segoe UI Emoji / Noto Color Emoji)
     - 미리보기·다운로드 결과에 그대로 포함
     - 드래그로 위치 이동
  ▪ 이모지 전용 편집 컨트롤
     - 선택 시 편집 패널이 이모지용으로 전환
     - 크기 슬라이더 (24~500px)
     - 빠른 위치 버튼(위·중앙·아래)
  ▪ 레이어 목록: 이모지 레이어는 실제 이모지를 아이콘으로 표시 + "이모지 스티커" 라벨


----------------------------------------
[3] 우상단 다운로드 영역 반응형 수정
----------------------------------------
요청:
  "우상단 다운로드 부분의 폰트 반응형이 제대로 안먹어 그부분 수정해줘"

원인:
  헤더가 justify-between flex인데 좌측 타이틀이 줄어들지 않아(shrink 미적용),
  화면이 좁아지면 우측 컨트롤(셀렉트·버튼)을 밀어내고 폰트가 고정된 채 깨짐.

수정 내용:
  ▪ 우측 컨트롤: 폰트·패딩 반응형 (모바일 text-xs / sm 이상 text-sm),
    shrink-0 고정, whitespace-nowrap, 초소형(420px 미만)에서는 아이콘만 표시
  ▪ 좌측 타이틀: min-w-0 + truncate 로 공간 부족 시 줄어들게 처리,
    로고 shrink-0, 타이틀 폰트 text-base sm:text-lg


----------------------------------------
[4] 디시인사이드 형태 디자인으로 변경
----------------------------------------
요청:
  "DC인사이드 형태의 디자인으로 변경가능해?"
  → 선택: "클래식 레트로" (돋움/굴림 12px, 얇은 회색 테두리,
           파란 제목바, 옛날 웹 버튼 등 디시 갤러리 특유의 빽빽한 BBS 룩)

변경 내용:
  ▪ 전체 톤 & 폰트
     - 본문 폰트 돋움/굴림 계열 12px, 글자색 #333
     - 바깥 배경 #e8e8e8, 중앙 흰색 갤러리 컨테이너(회색 테두리)
     - 둥근 모서리·그림자 제거 → 각진 테두리(#d2d2d2) 위주
  ▪ 상단 구조 (디시 갤러리 형태)
     - 네이비 GNB 바: DCINSIDE 워드마크 + 갈매기(SVG) 로고
     - 갤러리 헤더: 갈매기 뱃지 + "짤 생성기 갤러리" + PNG/JPG 셀렉트 + 💾 이미지 저장
     - 말머리 탭바: 전체 / 짤제작 / 이미지 / 텍스트
  ▪ 컴포넌트
     - 버튼: 회색 그라데이션 + #a8a8a8 테두리의 옛날 웹 버튼, 강조 버튼만 네이비/빨강
     - 카드: 파란 제목바(▪ 배경 이미지, ▪ 이모지 스티커 …)
     - 텍스트 레이어 목록: 게시판 글목록처럼 번호 + 행 hover + 구분선
     - 입력/셀렉트/컬러피커: 각진 테두리, 포커스 시 네이비
     - 이모지 그리드: 회색 박스, hover 시 연한 파랑
  ▪ 기능은 전부 그대로 유지
