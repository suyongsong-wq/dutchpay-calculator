────────────────────────────────────────
■ 1. 프로젝트 요청
────────────────────────────────────────
사용자:
  single-react-dev 와 single-server-specialist 두 에이전트가 힘을 합쳐
  [Server+AI] AI 별명 생성기를 제작해줘.
  1) 사용자가 이름, 성격, 취미 등을 입력하면
  2) AI(OpenAI API)가 재미있는 별명을 여러 개 생성해주고
  3) 마음에 드는 별명을 선택/복사할 수 있게 하기.
  - "귀여운 동물 이름 스타일로", "게임 캐릭터 느낌으로" 같은 톤/스타일 옵션 추가
  - 생성된 별명을 클립보드에 복사하는 기능 포함

Claude:
  두 에이전트가 동일한 API 규약에 맞춰 충돌 없이 작업하도록
  공통 사양을 먼저 정의하고 병렬로 실행함.

  [공통 API 규약]
   - POST /api/generate
   - 요청:  { name, personality, hobby, style }
   - 응답:  { nicknames: [ { name, reason } ] }   (별명 5개)
   - 서버는 process.env.OPENAI_API_KEY 사용, index.html 정적 서빙

  [역할 분담]
   - single-server-specialist → server.js + package.json
       (Express, 정적 서빙, OpenAI gpt-4o-mini 호출, API 키 환경변수 처리)
   - single-react-dev → index.html
       (React + Tailwind CDN, 입력 폼 + 스타일 선택 + 별명 카드 + 복사 버튼)

  ※ 두 에이전트 모두 Write 권한 거부로 멈췄으나, 사양이 명확했기에
    Claude가 직접 세 파일을 작성하고 문법 검증까지 완료함.

  [완성 파일]
   - server.js      : Express 서버, POST /api/generate → OpenAI 호출
   - index.html     : React 단일 페이지(입력/스타일/결과/복사)
   - package.json   : express 의존성, "npm start" 스크립트

■ 3. 서버 연결(실행)
────────────────────────────────────────
사용자:
  서버 연결해보자.

Claude:
  - npm install 로 express 설치, Node v24 확인(내장 fetch 사용 가능)
  - 포트 3000은 다른 node 프로세스(PID 87428)가 점유 중 → EADDRINUSE
  - 충돌을 피해 포트 3001로 실행 (PORT=3001 node server.js)

  [검증 결과]
   - GET /              → HTTP 200, index.html 정상 서빙
   - POST /api/generate → 키 미설정 시 안내 에러 정상 반환
                          (엔드포인트/라우팅/JSON 파싱 정상 동작)

  접속 주소: http://localhost:3001
  ※ 실제 AI 별명 생성은 OPENAI_API_KEY 설정 필요.

■ 4. 디자인 변경 - Windows XP UI
────────────────────────────────────────
사용자:
  디자인을 윈도우 XP UI 형태로 제작해줘.

Claude:
  기능/API 규약은 그대로 두고 UI만 클래식 Windows XP(Luna) 스타일로 교체.

  [적용 내용]
   - 바탕화면 : XP "Bliss" 느낌의 하늘색 → 초록 언덕 그라데이션
   - 창       : 파란 그라데이션 타이틀바 + (_ ▢ ✕) 캡션 버튼(닫기 빨강),
                베이지색(#ECE9D8) 본문
   - 폼       : XP 특유의 sunken 텍스트박스, 파란 라벨
   - 스타일   : fieldset 그룹박스 안 칩 버튼(선택 시 노란 하이라이트)
   - 버튼     : 은회색 그라데이션 + 호버 시 주황 글로우
   - 작업표시줄: 하단 고정, 초록 [⊞ 시작] 버튼 + 우측 실시간 시계
   - 폰트     : Tahoma

  서버 재시작 없이 정적 파일로 즉시 반영됨 (http://localhost:3001).
