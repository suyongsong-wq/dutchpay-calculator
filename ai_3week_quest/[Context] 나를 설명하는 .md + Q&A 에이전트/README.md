# 나를 소개하는 Q&A 봇 (Server + AI)

`나에_대한_추측.md`(나, suyong에 대한 소개 문서)를 **유일한 근거**로 질문에 답하는 1인 소개 챗봇입니다.
문서에 있는 내용만 답하고, **없는 내용은 "몰라요"**라고 답합니다. (AI의 일반지식이 아니라 '내 .md'를 근거로!)

## 구성

| 파일 | 역할 |
|------|------|
| `나에_대한_추측.md` | 봇이 답변의 근거로 쓰는 내 소개 (기본정보/경력/취향/좋아하는 것 섹션) |
| `server.js` | 백엔드. API 키를 서버에 숨기고, 나에_대한_추측.md를 시스템 프롬프트로 주입해 Claude를 호출 |
| `index.html` | 프론트엔드. React + Tailwind 채팅 UI |

## 실행 방법

1. Anthropic API 키 준비 (https://console.anthropic.com 에서 발급)
2. 키를 환경변수로 설정하고 서버 실행:
   ```bash
   ANTHROPIC_API_KEY=발급받은키 node server.js
   ```
3. 브라우저에서 접속:
   ```
   http://localhost:3000
   ```

> Node.js 18 이상이면 됩니다. 외부 패키지 설치(`npm install`) 불필요 — 내장 모듈만 사용합니다.

## API 계약

- `POST /api/ask`
  - 요청: `{ "question": "이 사람 무슨 일 해?" }`
  - 응답: `{ "answer": "..." }`
  - 에러: `{ "error": "한국어 메시지" }`

## 핵심 동작

- `server.js`가 시작될 때 `나에_대한_추측.md` 전체를 읽어 시스템 프롬프트의 `<about-me>...</about-me>`에 주입합니다.
- 시스템 프롬프트에 *"이 문서 내용만 근거로 답하고, 없으면 몰라요라고 해"*를 명시해 환각(추측)을 막습니다.
- 다른 사람을 소개하는 봇으로 바꾸고 싶으면 **`나에_대한_추측.md`만 교체**하면 됩니다.

## 확장 아이디어

`나에_대한_추측.md`만 바꾸면 → '친구 소개 봇', '우리 가게 소개 봇' 등으로 그대로 재사용할 수 있어요.
