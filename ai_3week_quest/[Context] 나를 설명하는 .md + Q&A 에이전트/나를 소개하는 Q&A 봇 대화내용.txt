==========================================================
[Server+AI] 나를 설명하는 .md + Q&A 에이전트 — 제작 대화 정리
날짜: 2026-06-15
==========================================================

■ 미션 (하버스쿨 AI 교육 / 3주차 퀘스트)
나에 대한 정보를 담은 .md 컨텍스트를 만들고, 그걸 근거로만 답하는
나만의 Q&A 봇 만들기.
  1) 내 이력·취향·경험을 정리한 about-me 컨텍스트(.md) 작성
  2) "이 사람 무슨 일 해?" 같은 질문 입력 받기
  3) 봇이 .md 내용을 근거로 답하고, 없는 내용은 "몰라요"라고 답하기
핵심: AI의 일반지식이 아니라 '내 .md'를 근거로만 답하게 하는 것.
협업: single-react-dev(프론트) + single-server-specialist(백엔드) 2명 협동.


■ 진행 순서 (대화 흐름)

1. 컨텍스트 소스 준비
   - 이전에 만들어 둔 '나에_대한_추측.md'(워크스페이스 분석으로 AI가 추측한
     suyong 프로필)를 봇의 근거 문서로 사용하기로 결정.
   - 처음엔 about-me.md(기본정보/경력/취향/좋아하는 것 섹션)를 새로 정리했다가,
     이후 "나에_대한_추측.md를 바탕으로 답하도록" 요청에 따라 근거 문서를
     '나에_대한_추측.md'로 교체하고 about-me.md는 정리(삭제).

2. 역할 분담 에이전트 2명 동시 실행
   - single-server-specialist → server.js (백엔드)
   - single-react-dev → index.html (프론트엔드)
   - 공통 API 계약 공유: POST /api/ask  { question } → { answer }

3. 백엔드(server.js) 구성
   - Node.js 내장 모듈만 사용(http/https/fs/path), 외부 의존성 0개.
   - 시작 시 '나에_대한_추측.md' 전체를 읽어 시스템 프롬프트의
     <about-me>...</about-me> 에 통째로 주입.
   - 시스템 프롬프트 규칙: "이 문서 내용만 근거로 답하고, 없으면 '몰라요'.
     일반 상식·추측 금지. 한국어로 친근·간결하게."
   - 모델: claude-opus-4-8 (Anthropic Messages API를 raw HTTPS로 호출).
   - API 키는 서버에만 숨김(process.env.ANTHROPIC_API_KEY). 프론트/코드에
     하드코딩 금지.

4. 프론트엔드(index.html) 구성
   - 단일 HTML, CDN 기반 React + Tailwind (빌드 도구 없음).
   - 따뜻한 크림/베이지 톤, 말풍선 채팅 UI, 봇 답변 타이핑 애니메이션.
   - 예시 질문 칩 4개("이 사람 무슨 일 해?" / "취미가 뭐야?" /
     "어떤 걸 만들었어?" / "나이가 어떻게 돼?"), Enter 전송(한글 조합 가드),
     로딩 인디케이터, 에러 말풍선.
   - 헤더에 "나에_대한_추측.md를 근거로만 답해요" + 📄 배지 표시.

5. 브라우저 확인 → 문제 발견
   - 포트 3000은 이전에 띄워둔 다른 [Server+AI] 앱이 점유 중이라 3456으로 변경.
   - 화면에서 질문하면 "서버에 API 키가 설정되지 않았어요" 에러 표시됨.
     (Claude 호출에 API 키가 필요하기 때문)

6. "MD에서 찾아서 답하도록" 추가 설정 (키 없이도 동작)
   - 키가 없을 때 '나에_대한_추측.md'에서 직접 찾아 답하는 '문서 검색 모드' 추가.
   - 한국어 조사 제거(예: "취미가"→"취미") + 자주 묻는 질문 의도 매핑으로
     문서의 해당 줄을 찾아 답변. 근거가 없으면 "몰라요".
   - 정리:  키 있음 → Claude 모드 / 키 없음 → 문서 검색 모드
            (어느 쪽이든 답은 오직 MD에서만 나옴)


■ 최종 동작 확인 (문서 검색 모드, 키 없이 테스트)
   Q: 이 사람 무슨 일 해?  → "…꼼꼼한 실용주의 학습자" / 하버스쿨 AI 교육 / "쓰면서 배운다"
   Q: 취미가 뭐야?         → 디자인·정리정돈·호기심 / 토스 스타일 시도 얘기
   Q: 어떤 걸 만들었어?     → 더치페이·세금 계산기, PDF·QR·짤 생성기, AI 꿈해몽 등
   Q: 이름이 뭐야?         → suyong
   Q: 사는 곳이 어디야?     → "몰라요" (문서에 없음)
   Q: 나이가 어떻게 돼?     → "실제 직업·나이·성별은 전혀 모름" (문서 근거)
   → 미션 충족: 문서에 있으면 그걸 근거로, 없으면 "몰라요".


■ 파일 구성
   나에_대한_추측.md   ← 봇의 유일한 근거 문서
   server.js          ← 문서를 읽어 답변(키 있으면 Claude, 없으면 문서 검색)
   index.html         ← React+Tailwind 채팅 UI
   package.json / .env.example / README.md

■ 실행 방법
   (키 없이 - 문서 검색 모드)  node server.js  → http://localhost:3000
   (키 있게 - Claude 모드)     ANTHROPIC_API_KEY=발급받은키 node server.js

■ 확장 아이디어
   '나에_대한_추측.md'만 다른 .md로 바꾸면 → '친구 소개 봇', '우리 가게 소개 봇'
   으로 그대로 재사용 가능.
