■ 과제 (미션)
수업에서 배운 내용을 질문하면 답변해주는 나만의 AI 에이전트 만들기.
  1) 텍스트 컨텍스트(강의 노트·개념)와 코드 컨텍스트(실습 코드)를 모두 참조
  2) 수업 내용을 질문하면 정확하게 답변
  3) 에이전트 메모리로 이전 대화 맥락을 기억

핵심 요구사항
  - 텍스트 + 코드 컨텍스트: 수업 자료와 실습 코드를 모두 참조할 수 있어야 함
  - 메모리 사용 필수: 이전 대화를 기억하고 맥락을 이어 답변


■ 최종 산출물 (두 가지 형태로 제작)
이 과제를 두 형태로 만들었습니다. 둘 다 동일한 컨텍스트(수업 자료)를 공유합니다.

  (A) 웹앱 버전  — 브라우저 채팅 UI + Node 서버
      위치: ai_3week_quest/[Context] 수업 내용 질문 에이전트/
  (B) Claude Code 에이전트 버전 — /agents 로 쓰는 서브에이전트
      위치: .claude/agents/class-qa.md
      (스크린샷이 안내한 /agents 방식 그대로)

■ 동작 방식 (요구사항 충족)
1) 텍스트 + 코드 컨텍스트 모두 참조
   - 서버가 시작될 때 context/ 폴더 전체(.md/.txt/.html/.js)를 읽어들임.
   - 로딩 결과: 텍스트 2개 + 코드 6개, 총 114,623자.
   - 화면의 "📚 참조 중인 수업 자료" 패널에서 어떤 자료를 보는지 직접 확인 가능.

2) 정확한 답변 (근거 기반)
   - "이 자료만 근거로 답하고, 없으면 솔직히 모른다고 말한다"는 Q&A봇 패턴 적용.
   - 개념 질문 → 지식베이스/노트에서 찾아 답하고 출처(파일·섹션) 표기.
   - "코드 보여줘" → context/code 의 실습 파일을 찾아 코드블록으로 보여줌.

3) 메모리 (필수)
   - 세션별 대화 기록을 서버가 보관 + memory/sessions.json 에 저장(재시작해도 유지).
   - 매 질문에 이전 대화를 함께 처리 → "아까 물어본 거랑 연결해서" 같은 후속 질문 처리.
   - 프론트는 localStorage 의 sessionId 로 같은 세션 유지, 새로고침 시 /api/history 로 복원.


■ 키 없이도 동작하는 하이브리드 (중요)
요청에 따라 API 키 없이도 돌아가도록 만들었습니다.
  - 키 없음(오프라인 모드): context/지식베이스.md 를 검색해 답변. (외부 호출 0, 무료)
  - ANTHROPIC_API_KEY 있음(AI 모드): Claude(claude-opus-4-8)로 더 풍부하게 답변.
  - 화면 상단 배지로 현재 모드 표시("🔌 오프라인 — 수업 자료 기반" / "🤖 AI 모드").
  - 키는 항상 서버에만 둠(수업에서 배운 "키 숨기기" 원칙). 프론트는 /api/... 만 호출.


■ API 계약
  GET  /                         index.html
  GET  /api/context              { textFiles, codeFiles, totalChars, mode, kbCount }
  POST /api/chat                 { sessionId, message } → { reply, sessionId }
  GET  /api/history?sessionId=   { messages:[{role,content}] }
  POST /api/reset                { sessionId } → { ok:true }
  (에러는 상태코드 + { error: "한국어 메시지" })

      · "더치페이 계산기 코드 보여줘"           → context/code 의 실습 파일 코드 출력
      · "아까 거랑 연결해서 더 자세히"          → 직전 주제로 이어서 답변(메모리 동작)
      · "점심 뭐 먹지?"(자료 밖)               → "수업 자료에는 안 나와요" + 주제 안내

  - 헤드리스 Chrome 으로 작업물 스크린샷 2장 캡처(초기화면 / 대화화면).
